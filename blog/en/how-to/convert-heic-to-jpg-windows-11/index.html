What is HEIC and Why Windows Doesn't Open It?

HEIC (High Efficiency Image Container) is a modern image format that Apple has been using in iOS since iOS 11 (2017). It's based on the HEVC (H.265) codec and provides better compression compared to JPG.

HEIC Advantages:

  • Smaller file size — 50% smaller than JPG at the same quality
  • Transparency support — like PNG, but more compact
  • 16-bit color depth — vs 8-bit in JPG
  • Live Photos storage — photo and video in one file

Why Windows Doesn't Support HEIC?

Windows 10 and 11 can open HEIC files, but require installing the free "HEIF Image Extensions" from Microsoft Store. Without it, you'll see an error:

"Windows can't open this file. Choose a program to open this file."

Many users prefer to convert HEIC to universal JPG so photos open everywhere without additional software.

Method 2: Built-in Windows 11 Tools

Windows 11 can convert HEIC to JPG using built-in tools, but the process isn't very convenient.

Method 2.1: Via Photos App

  1. Install "HEIF Image Extensions" from Microsoft Store (free)
  2. Open HEIC file in Photos app
  3. Click "Edit" → "Save a copy"
  4. When saving, select JPG format

Method 2.2: Via Paint

  1. Install "HEIF Image Extensions"
  2. Open HEIC file in Paint (right-click → Open with → Paint)
  3. Click "File" → "Save as" → "JPEG picture"

Disadvantages of this method:

  • Need to convert each file manually
  • No batch processing
  • Time-consuming for many photos
  • May lose quality when saving in Paint

Suitable for: Converting 1-2 photos when you don't want to install additional software.

How to Change Photo Format on iPhone

If you don't want to convert HEIC to JPG every time, configure your iPhone to save photos in JPG format.

Method 1: Automatic Conversion on Transfer

  1. Open Settings on iPhone
  2. Scroll down to Photos
  3. Under "TRANSFER TO MAC OR PC" select "Automatic"

Method 2: Completely Disable HEIC

  1. Open Settings → Camera
  2. Tap Formats
  3. Select "Most Compatible" instead of "High Efficiency"
